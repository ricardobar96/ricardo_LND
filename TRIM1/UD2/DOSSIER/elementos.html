Elementos HTML


Repasando los conceptos de HTML



A continuación mostraremos algunos de los elementos aprendidos a lo largo del curso:


Elemento Estado Descripción
<link>
Introducido
HTML Internet Draft 1.2
Especifica relaciones entre el documento actual y otro recurso externo.
<meta>
Introducido
Mencionado en la primera especificación de HTML2​, especificado en HTML Internet Draft 1.2
Representa información sobre el propio documento HTML, es decir, metadatos
<style>
Introducido
HTML 3.2
Contiene estilos para el documento, casi siempre, CSS
<title>
Introducido
HTML 1.0
Define el título del documento
